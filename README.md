InvoiceSpider是一个可以自动化模拟在[国家税务总局全国增值税发票查验平台](https://inv-veri.chinatax.gov.cn)查验发票的项目。

## 停止更新

2020年4月以来，查验平台进行了多次更新，增加了多个加密参数，本人是在校学生，精力有限，查验平台迭代速度太快，我慎重考虑之后决定停止更新这个项目，在这里致敬关注和帮助过我的伙伴们。
